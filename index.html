Wij gaan het hebben over verschillende onderwerpen maar eerst over het Ram geheugen. RAM staat voor Random Access Memory en wordt wel werkgeheugen genoemd. RAM slaat alle informatie op van de processor, hoe meer RAM geheugen hoe sneller de computer werkt. Waar je de computer snel voor nodig hebt wordt veel geheugen aangeraden, bijvoorbeeld: Voor gamen minimaal 16 GB aanbevolen en voor foto's editen minder RAM, daar wordt 8 GB aanbevolen, omdat de computer minder snel hoeft te werken. Je kan het RAM geheugen uit de computer halen als een los onderdeel om te vervangen of te repareren. Het RAM geheugen bestaat uit twee modules, DDR3 en DDR 4. Het verschil tussen de twee is dat DDR 3 minder zuinig is dan DDR 4. DDR 4 is ook altijd wat duurder dan DDR 3.

Wat is een moederbord en waarom heet het een moederbord?

Een moederbord voorziet alle componenten van stroom en laat deze met elkaar communiceren. Het moederbord bevat de centrale processor, die bepaalt hoe snel je PC informatie kan verwerken. Je processor, werkgeheugen, geluidskaart, videokaart en andere onderdelen zijn dus allemaal aangesloten op het moederbord. Een moederbord is een printplaat die componenten van de computer samen verbind via transistoren en met elkaar laat communiceren.
