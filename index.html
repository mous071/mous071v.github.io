Inleiding: Welkom op onze website! Wij gaan het hebben over RAM geheugen. RAm staat voor Random Acces Memory en wordt wel werkgeheugen genoemd. RAM slaat alle informatie op van de processor, hoe meer RAM geheugen hoe sneller de computer werkt. waar je de computer snel voor nodig hebt wordt veel geheugen aangeraden, bijvoorbeeld: voor gamen voor minimaal 16 GB aanbevolen en voor Foto's minder daar wordt 8 GB aanbevolen, omdat de computer minder snel hoeft te werken.
