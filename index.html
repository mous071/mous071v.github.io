Wij gaan het hebben over RAM geheugen. RAM staat voor Random Acces Memory en wordt wel werkgeheugen genoemd. RAM slaat alle informatie op van de processor, hoe meer RAM geheugen hoe sneller de computer werkt. waar je de computer snel voor nodig hebt wordt veel geheugen aangeraden, bijvoorbeeld: voor gamen voor minimaal 16 GB aanbevolen en voor Foto's editen minder, daar wordt 8 GB aanbevolen, omdat de computer minder snel hoeft te werken.

Wat is een moederbord en waarom heet het een moederbord?

Een moederbord voorziet alle componenten van stroom en laat deze met elkaar communiceren. Het moederbord bevat de centrale processor die bepaalt hoe snel je PC informatie kan verwerken. Je processor, werkgeheugen, geluidskaart, videokaart en andere onderdelen zijn dus allemaal aangesloten op het moederbord. (Een moederbord (in een personal computer; ook wel systeembord of mainboard) is een printplaat met elektronica waarop een aantal andere (insteek-) printplaten gemonteerd kunnen worden (vandaar de naam) . In de loop der jaren is steeds meer functionaliteit op het moederbord ondergebracht.
