RAM staat voor Random Access Memory en wordt wel werkgeheugen genoemd. RAM slaat alle informatie op van de processor, hoe meer RAM geheugen hoe sneller de computer werkt. Waar je de computer snel voor nodig hebt wordt veel geheugen aangeraden, bijvoorbeeld: Voor gamen minimaal 16 GB aanbevolen en voor foto's editen minder RAM, daar wordt 8 GB aanbevolen, omdat de computer minder snel hoeft te werken. Je kan het RAM geheugen uit de computer halen als een los onderdeel om te vervangen of te repareren. Het RAM geheugen bestaat uit twee modules, DDR3 en DDR 4. Het verschil tussen de twee is dat DDR 3 minder zuinig is dan DDR 4. DDR 4 is ook altijd wat duurder dan DDR 3.
