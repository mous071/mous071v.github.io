Ram en ROM geheugen.

RAM staat voor Random Access Memory en wordt wel werkgeheugen genoemd. RAM slaat alle informatie op van de processor, hoe meer RAM geheugen hoe sneller de computer werkt. Waar je de computer snel voor nodig hebt wordt veel geheugen aangeraden, bijvoorbeeld: Voor gamen minimaal 16 GB aanbevolen en voor foto's editen minder RAM, daar wordt 8 GB aanbevolen, omdat de computer minder snel hoeft te werken. Je kan het RAM geheugen uit de computer halen als een los onderdeel om te vervangen of te repareren. Het RAM geheugen bestaat uit twee modules, DDR3 en DDR 4. Het verschil tussen de twee is dat DDR 3 minder zuinig is dan DDR 4. DDR 4 is ook altijd wat duurder dan DDR 3. Het ROM geheugen staat voor Read Only Memory oftewel het geheugen dat door de computer gelezen wordt. Het ROM geheugen is noodzakelijk voor de computer voor de UEFI instellingen, een soort vroegere BIOS voor de computer. ROM is een permanente geheugen dus niet net als RAM die maar tijdelijk is. Hoeveel RAM geheugen je nodig hebt hangt er vanaf wat je met de computer gaat doen. Doe je veel aan mailen, Youtube of Netflix? Dan is 4 GB RAM geheugen genoeg. Ga je echt gamen of fotobewerking dat zit je wel rond de 8 รก 16 GB aan RAM dat je nodig hebt. Het RAM geheugen wordt in tegenstelling van ROM geheugen gewist als de stroom van de computer opeens afgaat.
