3 lagenmodel

Deze pagina gaat over het 3 lagenmodel. Bij een ICT apparaat is sprake van lagen in het apparaat. De 3 lagen bestaan uit: Toepassingenlaag, oftewel application layer, logische laag, ook wel systeemsoftware, en de fysieke laag, de hardware van de computer. Tekstverwerkers zoals Docs en Word zijn voorbeelden van de toepassingenlaag, het moderbord is een voorbeeld van een fysieke laag, loische laag zijn besturingssystemen zoals Windows of Linux. Er bestaan 3 interfaces voor de gebruiker, GUI, graphical user interface, VUI, voice user interface, CLI, command line interface. Drivers zorgen ervoor dat de muis van een computer goed kan werken. In de menu vind je de groene knop, daar leggen we meer uit over het andere onderwerpen.
