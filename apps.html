Voorbeelden van apps zijn Youtube, Photoshop, Excel of Clash of Clans. Je downloadt apps van de Google Play Store of de Apple Store. Door de fabrikant zijn standaard enkele apps ge√Ønstalleerd. Bij games die gratis te spelen zijn kan je altijd iets kopen dat je beter maakt in het spel, dat noem in app aankopen. Er bestaan verschillende soorten apps, zoals een Native app. Dat is een mobiele app die je moet installeren om er gebruik van te kunnen maken. Een webapp is een app maar dan via een browser zoals Chrome. Door HTML kan je dan via een browser de app gebruiken, zoals Gmail. Die kan je downloaden via de stores, maar als je snel wilt zijn dan kan je ook via de website gebruik daarvan maken. Een hybride app is een mix tussen HTML apps en Natvie apps. Voorbeeld is de Facebook app, die is op Android en Apple hetzelfde, maar in de code anders.
