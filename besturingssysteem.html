Besturingssystemen

Deze pagina gaat over besturingssystemen. Wat is een besturingssysteem? Een besturingssysteem afgekort OS, operating system, is een groep programma's bij elkaar die na het opstarten het hardware aanstuurt en zorgt er voor dat je apps kan opstarten en beëindigen. Zonde rde software kan je de computer niet gebruiken. De meeste laptops draaien op Chrome OS of op macOS, maar de meest bekende is Windows. Microsoft Windows is een OS die in 1985 is uitgebracht en sinsdien hét besturingssysteem is van elke computer. Tegenwoordig is Windows 10 en 11 de besturingssystemen waar de computers momenteel op draaien. Windows heeft 2 licenties, share source en closed source. Closed source betekent dat de broncode niet vrijgegeven is voor de consument. Share source betekent dat je de broncode deelt met de consument. Operating Systems worden altijd geïnstalleerd op een SSD of een HDD. Het kan ook zijn dat een OS wordt geïnstalleerd of op een USB stick of cd. Een applicatie kan worden gebruikt door middel van een API, application programming interface. Een API geeft toegang tot de verschillende randapparatuur zoals HDD en beeldscherm. De hoofdtaken van een OS zijn het opstarten van de systeem, het toegang geven tot RAM geheugen en printer, communicatie van randapparatuur en programma's uitvoeren. De processor voert die taak uit.
